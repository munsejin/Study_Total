****************************
*  KESS 3.1 버전 설치방법  *
****************************

KESS(Korean Educational Statistics Software)는 엑셀 고유의 분석 기능에 VBA를 이용하여 개발된 
통계분석기능을 추가하여 사용할 수 있도록 만든 통계자료분석용 소프트웨어이다. 엑셀은 사회에서
기본적인 자료분석에 많이 활용되는 프로그램이나 회계 업무를 위해 개발된 관계로 제공되는 통계
자료분석 기능은 매우 제한적이다. 통계교육에 필요한 분석기법과 사회업무에서 유용하게 사용될 수
있는 자료분석 기능들을 추가하여 교육 및 업무에 활용할 수 있도록 개발하였다. 


KESS는 다음 주소에서 다운로드 받을 수 있다.

   http://stat.snu.ac.kr/time/kess_main.html 


1. KESS2013.zip의 구성파일

   KESS.xlam    : KESS의 시작 파일
   Kess1.xlam ~ Kess10.xlam  :  KESS의 구성 파일
   KESS2013.chm    : KESS의 도움말 파일
   DATA        : KESS의 예제용 데이터 파일 디렉토리
   README.TXT  : KESS의 설명 파일


2. KESS 설치법

   1단계: KESS를 다운로드 받을 폴더를 만든다. (예. C:\KESS)
   2단계: 다운 받은 Kess2013.zip 파일을 더블 클릭하여 압축을 풀어 지정된 폴더에 저장한다.(예. C:\KESS)
   3단계: 바탕화면에 KESS의 바로가기 아이콘을 만든다. 
          KESS.xlam 파일을 선택하고 마우스의 오른쪽 버튼을 눌러 
          '보내기 - 바탕 화면에 바로 가기 만들기'를 선택하면 바로가기 아이콘이 바탕화면에 만들어진다.

3. KESS의 구동
   KESS 바로가기를 더블 클릭하면 엑셀이 구동되면서 KESS.xlam 매크로가 포함되어 있다는 대화상자가 나타나고, 
          매크로 포함(E)을 선택하면 KESS가 구동된다.

참고 : 엑셀이 구동되면서 자동으로 KESS를 실행하려면 다음과 같이 엑셀의 추가기능을 이용하면 된다.
          
   1) 엑셀의 파일 메뉴에서 옵션을 클릭하면 엑셀옵션 대화상자가 나타난다.
   2) 추가기능을 선택하면 엑셀에서 사용할 수 있는 추가기능들이 나열되고 화면 맨 아래 줄에
      관리(A)와 함께 리스트 상자와 이동(G)이라는 상자가 나타난다.
   3) 이동을 클릭하면 추가기능 대화상자가 나타나는데 여기서 찾아보기(B)를 이용하여 KESS.Xlam을 
      추가하면 된다.

4. KESS 사용시 유의사항

   1) KESS의 구성 파일이 손상된 경우에는 KESS를 다시 설치해야 한다.
   2) KESS를 이용하여 [ 통계분석 ]을 수행하려면 워크시트에 분석할 자료가 들어있어야 한다. 
      분석할 자료가 들어있는 워크시트나 KESS가 구동된 상태에서 외부 파일을 불러들인 후 
      [통계분석]을 눌러서 원하는 분석을 시작한다. 분석할 자료가 없는 경우, 에러 메시지가 출력된다.
   3) 예제용 데이터 파일을 보존하기 위해서는 사용한 후에 변경된 내용을 저장하면 안된다.
   4) KESS는 엑셀의 VBA(Visual Basic for Applications)에 의한 매크로로 작성된 프로그램이다.
      따라서 KESS.XLAM를 더블클릭하거나, 바탕화면에서 KESS 아이콘을 더블클릭하면 엑셀이 구동되면서 
      [매크로 제외(D)]와 [매크로 포함(E)]의 두 가지 옵션 중에서 하나를 선택하는 화면이 나타난다. 
      여기서 [매크로 포함(E)]을 선택해야만 KESS가 구동된다. 만약 [매크로 제외(D)]를 선택하면 
      엑셀만이 구동되어 KESS를 사용할 수 없다. 
   5) 엑셀의 2007 이전의 하위버전을 사용할 경우 문제가 생길 수 있다.
